Qunix OS

A modern Unix-like operating system written in Rust, designed for performance, stability, and simplicity.

View on GitHub

Kernel Architecture

Rust-based kernel with full process model, signals, and memory management.

Filesystem Support

Unified VFS with ext4, FAT32, tmpfs, devfs, and procfs support.

Scheduler

Preemptive, priority-driven scheduler designed for multi-core systems.

Interprocess Communication

Efficient IPC with pipes, shared memory, and futex mechanisms.

Userland

Includes 70+ utilities with a POSIX-compatible shell environment.

Performance

Optimized for fast boot times and minimal system overhead.
